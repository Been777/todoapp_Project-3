## 👋 프로젝트 소개
회원가입, 로그인 기능이 있는 투두앱 백엔드 서버 만들기


## 🛠️ 개발 환경
- 언어: Java
- 버전: JDK 17
- 개발 환경: IntelliJ


## 🦻🏻 주요 기능
- 일정 (등록, 수정, 조회, 삭제)
- 댓글 (등록, 수정, 삭제)
- 로그인, 회원가입
